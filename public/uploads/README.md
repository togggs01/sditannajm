# Folder Uploads

Folder ini digunakan untuk menyimpan file yang diupload melalui admin panel, seperti:
- Foto guru
- Foto berita
- Gambar galeri
- Dan file lainnya

## Catatan
- File yang diupload akan disimpan dengan format: `timestamp-namafile.ext`
- Folder ini di-ignore di git (kecuali file .gitkeep dan README.md)
- Pastikan folder ini memiliki permission write untuk aplikasi
